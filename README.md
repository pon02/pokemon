# 反省点
ListとMapの二つを使用するプロジェクトを作成したかったため無理やりこの形にしたが、実際はMapのみで書ける内容になった。

ListからMapを作成するにはもっと内容を練る必要があった。

表示したい実行結果を出すために、try~catchの文を3回使用したが、大きなプロジェクトでは例外処理もはるかに多いと思うのでもっと簡潔に記載する方法があると思われるが、まだ調べきれていない。
